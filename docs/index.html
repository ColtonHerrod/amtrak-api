This site is archived

This website previously built itself automatically from Amtrak data, updated about every 20 minutes. However, the Amtrak data source is remarkably flaky and for a hobby project, it isn't worth the amount of naggy emails I get from the automatic build process whenver Amtrak delivers bad data.

The source code used to generate this site – and the data files that it hosted – is available for free on GitHub. Please feel free to grab it and do with it as you please! If you're curious why I built this thing in the first place, I wrote a blog post about it in 2023.
